# Retail Profit Intelligence Dashboard

## Project Overview
This project is an interactive Power BI dashboard developed to analyze retail sales performance, profitability, discount impact, and regional business trends.

The dashboard provides valuable business insights to help identify:
- High-performing product categories
- Loss-making sub-categories
- Monthly sales & profit trends
- Discount impact on profitability
- Global sales distribution
- Business improvement opportunities

---

# Tools & Technologies Used
- Python
- Pandas
- Power BI
- DAX
- CSV Dataset

✅ Discount Impact Analysis

---

# Key Business Insights
- Technology category generated the highest profit.
- Tables sub-category showed consistent losses.
- Q4 recorded the highest sales growth.
- High discounts negatively affected profitability.
- North America contributed the highest sales.

---

# Business Recommendations
- Reduce excessive discounts on loss-making products.
- Increase focus on high-profit product categories.
- Prepare inventory for seasonal demand spikes.
- Optimize low-margin sub-categories for profitability.
- Monitor high-discount products regularly.
